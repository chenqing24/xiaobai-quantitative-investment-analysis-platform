# xiaobai-quantitative-investment-analysis-platform

看书后自己实现的小白量化投资分析平台的实现。
优化部分过期代码，补充了注释。
Mac，Python 3.11，Vscode下运行测试通过。

## 参考

* 《零基础搭建量化投资系统》（作者：何战军、杨茂龙、何天琦）的第九章

## 功能

* 数据获取：通过API、爬虫和文件导入
* 数据分析：
  * 股价和成交量分析趋势
  * 构造自定义指标
  * 根据股票基本信息、行业数据、政策面来筛选股票和预测
  * 多因子模型
  * 深度学习模型
* 策略定制
* 历史回测
* 策略执行：API和外挂2种方式
